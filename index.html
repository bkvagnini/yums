YUMS

YUMS (or Your URL Management System) is a self hosted replica of Delicious, minus the social aspects. It is authored in PHP, but with the ambitious goal of using JSON as the database, since the actual searches don't require any major TransAct-SQL.

Project steps:

  • designing the UI
  • creating the project
  • testing it
  • figuring out an installer for it
  • developing screenshots/documentation for it
